# ZKTime 5.0 - Build 153 (ZKTeco Attendance Software)

ZKTeco biometric attendance management software - ZKTime 5.0 version 153.

## Contents
- `build153_ZKTeco/setup.exe` - Main installer
- `build153_ZKTeco/files/` - Application files (Att.exe, DLLs, reports)
- `build153_ZKTeco/driver/` - Device drivers
- `build153_ZKTeco/USBDriver/` - USB drivers (x86 & x64)

## Installation
1. Run `build153_ZKTeco/driver/setup.exe` first to install drivers
2. Then run `build153_ZKTeco/setup.exe` to install ZKTime 5.0
3. Connect your ZKTeco biometric device via USB or network

## Compatibility
- Windows OS
- ZKTeco fingerprint/face recognition devices

## Notes
- For USB devices, install USB driver from `USBDriver/` folder
- Supports SQL Server database backend
